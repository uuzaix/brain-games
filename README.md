# Console arithmetical games in scala

## Examples

### Game "Is number even"
```
Answer "yes" if number is even, otherwise answer "no". Answer "stop" to stop the game.
Question: 23
Your answer: no
Correct!
Question: 57
Your answer: yes
Wrong!
Question: 14
Your answer: foo
Let's try again
Question: 64
Your answer: stop
The game is stopped
```
